## 1 Java程序员充电、求职必备的核心知识库

全方位详细深入阐述高级Java程序员必备的软件设计技能。按照现有计划，主要研究如下方面知识点：

### 软件设计原则
1. 开闭原则
2. 依赖倒置原则
3. 单一职责原则
4. 接口隔离原则
5. 迪米特法则
6. 里氏替换原则
7. 合成复用原则

### 设计模式
1. 简单工厂
2. 工厂方法模式
3. 抽象工厂模式
4. 建造者模式
5. 单例模式
6. 原型模式
7. 外观模式
8. 装饰者模式
9. 适配器模式
